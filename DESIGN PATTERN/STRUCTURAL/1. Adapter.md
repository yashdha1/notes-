Creating adapter between the classes for seamless integration between the application. You can create an _adapter_. This is a special object that converts the interface of one object so that another object can understand it.

![[Pasted image 20260304214808.png]]

Adapter Pattern abstracts the complexities of another object for it to seamlessly integrate with other object. 
Adapters can not only convert data into various formats but can also help objects with different interfaces collaborate. Here’s how it works:
1. The adapter gets an interface, compatible with one of the existing objects.
2. Using this interface, the existing object can safely call the adapter’s methods.
3. Upon receiving a call, the adapter passes the request to the second object, but in a format and order that the second object expects.

> "Convert the interface of a class into another interface that clients expect."


### Example 1
Imagine you have an old system that speaks "XML" but a new system that speaks "JSON":

```python
# Existing class (Adaptee) - speaks XML
class XmlDataProvider:
    def get_xml_data(self):
        return "<data><name>Alice</name><age>30</age></data>"


# Target interface the client expects
class JsonDataProvider:
    def get_json_data(self):
        raise NotImplementedError


# Adapter - wraps XmlDataProvider and exposes JSON interface
class XmlToJsonAdapter(JsonDataProvider):
    def __init__(self, xml_provider: XmlDataProvider):
        self._xml_provider = xml_provider

    def get_json_data(self):
        xml = self._xml_provider.get_xml_data()
        # Simplified conversion for illustration
        return {"name": "Alice", "age": 30}  # parsed from XML


# Client code - only knows about JsonDataProvider
def client(provider: JsonDataProvider):
    data = provider.get_json_data()
    print(f"Got JSON: {data}")


# Usage
xml_provider = XmlDataProvider()
adapter = XmlToJsonAdapter(xml_provider)
client(adapter)
# Output: Got JSON: {'name': 'Alice', 'age': 30}
```

### Key Roles

| Role        | In Example                                 |
| ----------- | ------------------------------------------ |
| **Adaptee** | `XmlDataProvider` (existing, incompatible) |
| **Target**  | `JsonDataProvider` (what client expects)   |
| **Adapter** | `XmlToJsonAdapter` (the bridge)            |
| **Client**  | `client()` function                        |

### When to Use

- Integrating **legacy code** with new systems
- Using **third-party libraries** that don't match your interface
- Making **incompatible classes** collaborate without modifying their source

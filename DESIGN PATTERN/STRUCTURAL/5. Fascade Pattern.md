
The **Facade** pattern provides a **simplified interface** to a complex subsystem.
### Concept
> "One simple interface hides a complex system underneath." Hides complicated processes underneth a Fascade. 

![[Pasted image 20260304232837.png|390]]
### Real-World Analogy
A **TV remote** — you press "ON" and it handles power, input signal, display calibration, sound system... you don't care about the complexity inside.
### Example

```python
# --- COMPLEX SUBSYSTEMS ---
class DVDPlayer:
    def on(self): print("DVD Player ON")
    def play(self, movie): print(f"Playing '{movie}'")
    def off(self): print("DVD Player OFF")

class Projector:
    def on(self): print("Projector ON")
    def set_input(self, source): print(f"Projector input set to {source}")
    def off(self): print("Projector OFF")

class SoundSystem:
    def on(self): print("Sound System ON")
    def set_volume(self, level): print(f"Volume set to {level}")
    def off(self): print("Sound System OFF")

class Lights:
    def dim(self, level): print(f"Lights dimmed to {level}%")
    def on(self): print("Lights ON")


# --- FACADE - one simple interface ---
class HomeTheaterFacade:
    def __init__(self):
        self._dvd      = DVDPlayer()
        self._projector = Projector()
        self._sound    = SoundSystem()
        self._lights   = Lights()

    def watch_movie(self, movie: str):
        # functionality 
        print("--- Getting ready to watch movie... ---")
        self._lights.dim(20)
        self._projector.on()
        self._projector.set_input("DVD")
        self._sound.on()
        self._sound.set_volume(15)
        self._dvd.on()
        self._dvd.play(movie)

    def end_movie(self):
	    # end_movie wali functionality
        print("\n--- Shutting down theater... ---")
        self._dvd.off()
        self._sound.off()
        self._projector.off()
        self._lights.on()


# --- Client - only talks to the Facade ---
theater = HomeTheaterFacade()
theater.watch_movie("Inception")
theater.end_movie()
```

### Output

```
--- Getting ready to watch movie... ---
Lights dimmed to 20%
Projector ON
Projector input set to DVD
Sound System ON
Volume set to 15
DVD Player ON
Playing 'Inception'

--- Shutting down theater... ---
DVD Player OFF
Sound System OFF
Projector OFF
Lights ON
```

### Key Roles

|Role|In Example|
|---|---|
|**Facade**|`HomeTheaterFacade`|
|**Subsystems**|`DVDPlayer`, `Projector`, `SoundSystem`, `Lights`|
|**Client**|only calls `watch_movie()` / `end_movie()`|

### When to Use

- A system is **too complex** to expose directly
- You want a **simple entry point** to a library or framework
- You need to **layer your architecture** cleanly

#### starlett 
- *Starlette* is a *lightweight Python web framework* designed for building **high-performance async web services and APIs**. It is built *directly on top of the ASGI standard*, which is the modern replacement for WSGI used in frameworks like Django or Flask. 
- It is **very minimal and fast** and is often used as the **foundation for larger frameworks**. 
- FastAPI (FastAPI internally uses Starlette).
- *ASGI (Asynchronous Server Gateway Interface)** is a **standard interface between Python web apps and web servers.*


**Uvicorn** :
```
Uvicorn is a fast, lightweight Python web server designed to run asynchronous Python web applications. It implements the **ASGI** standard, which is the modern successor to **WSGI**.
```
Python web apps need **a server process** that:
1. Listens for HTTP requests (port 80 / 8000 etc.)
2. Passes the request to your Python app
3. Gets the response
4. Sends it back to the client
```
Browser → Uvicorn → Python App → Uvicorn → Browser
```
